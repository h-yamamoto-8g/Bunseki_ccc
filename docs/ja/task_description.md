# タスクについて

## タスクとは

環境分析課では、排水や環境水の分析依頼を受け、サンプルの分析から結果の報告までを日常的に行っています。この一連の業務の流れを、本アプリでは「タスク」として管理します。

たとえば「4月6日のCOD分析」や「今週の重金属分析」のように、ある分析項目について依頼を受けてから結果を報告するまでの一連の作業が、1つのタスクに該当します。タスクには分析対象のサンプル情報、分析時のチェック記録、入力した結果データ、確認者への回覧状況など、その業務に関わるすべての情報が紐づきます。

担当者はタスクを起票し、定められた手順に沿ってステートを1つずつ進めていきます。各ステートでは、その段階で必要な作業（サンプルの確認、分析前後のチェック、データ入力、結果の確認、上長への回覧など）を行います。すべてのステートが完了すると、そのタスクは「終了」となり、業務が完結します。

このように、タスクとは「1回の分析業務を最初から最後まで追跡・記録するための単位」です。誰が、いつ、どのサンプルを、どのように分析し、誰が確認したのかという業務の履歴が、タスクを通じて一元的に管理されます。

---

## タスクの進め方

タスクは7つのステートで構成されています。担当者は起票から順に各ステートを進め、最終的に確認者の承認を経て業務を完了させます。

```
起票 → サンプル → 分析 → 入力 → チェック → フロー → 終了
```

### 起票

業務の出発点です。担当者は、どの分析項目（ホルダーグループ）を対象とするかを選び、関連するJOB番号を紐づけてタスクを作成します。ここで指定した分析項目に応じて、以降のステートで表示されるサンプルやチェックリストの内容が決まります。

### サンプル

起票した分析項目に対して、今回分析するサンプルの一覧を確認します。システムが自動的にサンプルを抽出しますが、必要に応じてサンプルの追加や除外を行うこともできます。ここで確定したサンプルが、以降の分析・入力・確認の対象となります。

### 分析

実際の分析作業を行うステートです。分析の前後に確認すべき事項がチェックリストとして表示されるので、マニュアルや関連資料を参照しながら1項目ずつ確認を進めます。分析前のチェック（器具の準備、試薬の確認など）と分析後のチェック（片付け、記録の確認など）をすべて完了させることで、次のステートに進めます。

### 入力

分析で得られた結果データをシステムに登録するステートです。データの入力はLab-Aid（外部ツール）を通じて行います。Lab-Aidでの入力が完了したら、データ更新を実行して本アプリにデータを取り込みます。

### チェック

入力された結果データに問題がないかを確認するステートです。各サンプルの分析値が一覧で表示され、過去のデータと比較して異常がないかを確認します。異常値がある場合はフラグが表示されるほか、トレンドグラフで過去の推移を視覚的に確認することもできます。すべての確認項目を完了させることで、回覧に進めます。

### フロー

分析結果を上長や関係者に回覧し、承認を受けるステートです。担当者は確認者を指定して回覧を開始します。確認者はメールで通知を受け、結果を確認したうえで次の確認者に転送するか、問題があれば担当者に差し戻します。最終確認者が承認すると、タスクは完了となります。回覧中はコメントやファイルの添付を通じて、担当者と確認者の間でやりとりを行うことができます。

### 終了

すべての確認が完了し、タスクが完結した状態です。タスクの概要（分析項目、サンプル数、作成者、完了日時など）が表示されます。終了したタスクは編集できませんが、記録として参照することができます。

---

## タスクのステータス

タスクは進行状況に応じて、以下の3つのステータスで管理されます。

**進行中** — 担当者が各ステートの作業を進めている段階です。起票からフロー送信前までがこの状態にあたります。差し戻しによって回覧から戻ってきたタスクも、再び「進行中」になります。

**回覧中** — 結果が確認者に送られ、承認を待っている段階です。確認者が順番に確認を進めている間、この状態が続きます。

**終了** — 最終確認者の承認が完了し、業務が完結した状態です。

---

## ステートの手戻りについて

業務の途中で前のステートの内容を修正する必要が生じた場合、過去のステートに戻って編集することができます。ただし、あるステートを編集すると、それ以降のステートで記録した内容はリセットされます。これは、前提となる情報が変わった以上、後続の作業もやり直す必要があるためです。手戻りの際には確認画面が表示されるので、意図しないリセットを防ぐことができます。
